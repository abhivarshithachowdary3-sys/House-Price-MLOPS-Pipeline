# House-Price-MLOPS-Pipeline

An end-to-end machine learning pipeline developed using Python and Docker to automate the complete model development workflow. The project includes configurable data preprocessing, model training, evaluation, logging, and testing while ensuring reproducibility and easy deployment.

## Features

* End-to-end machine learning workflow
* Configurable training through YAML configuration
* Automated data preprocessing
* Regression model training and evaluation
* Performance metrics generation
* Structured logging
* Unit testing support
* Dockerized execution
* Reproducible pipeline

## Tech Stack

* Python
* Scikit-learn
* Pandas
* NumPy
* PyYAML
* Docker
* Pytest
* Logging

This project demonstrates MLOps best practices by combining reproducible machine learning workflows, automated testing, containerization, and configuration-driven execution.
